**Basic HTML Project**

This is a beginner-friendly HTML project developed using Visual Studio Code. It demonstrates the foundational structure and elements of a static web page, ideal for learning and practicing core HTML concepts.

 **Project Features**

- Semantic HTML layout (<header>, <main>, <footer>)
- Basic text formatting using headings, paragraphs, and lists
- Image embedding and hyperlinking
- Simple form elements (e.g., input, button)
- Clean and readable code structure
  
   **Purpose**
  
This project was created to:
- Strengthen my understanding of HTML syntax and structure
- Serve as a reference for future web development projects
- Help beginners explore how HTML works in a real-world editor like VS Code
